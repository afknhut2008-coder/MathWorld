📐 CHEAT SHEET - TÍCH PHÂN

📝 MẪU CÚ PHÁP NHẬP HÀM ▼
Lũy thừa
x^n → xⁿ (VD: x^2, x^3, x^(1/2))
Căn bậc hai
sqrt(x) → √x
Hàm mũ
exp(x) → eˣ  |  e^x → eˣ
Logarit tự nhiên
log(x) → ln(x)
Logarit cơ số 10
log10(x) → log₁₀(x)
Lượng giác
sin(x), cos(x), tan(x)
Hàm trị tuyệt đối
abs(x) → |x|
Hằng số
pi → π  |  e → e
Ví dụ mẫu
x^2 + 2*x + 1
sin(x)*cos(x)
exp(-x^2)
1/(1+x^2)
sqrt(1-x^2)
🔵 CÔNG THỨC ĐẠO HÀM ▼
\((x^n)' = n x^{n-1}\)
\((\sin x)' = \cos x\)
\((\cos x)' = -\sin x\)
\((\tan x)' = \frac{1}{\cos^2 x}\)
\((e^x)' = e^x\)
\((\ln x)' = \frac{1}{x}\)
\((u \pm v)' = u' \pm v'\)
\((uv)' = u'v + uv'\)
\(\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}\)
🟢 NGUYÊN HÀM ▼
\(\int x^n dx = \frac{x^{n+1}}{n+1} + C\)
\(\int \frac{1}{x} dx = \ln|x| + C\)
\(\int \sin x dx = -\cos x + C\)
\(\int \cos x dx = \sin x + C\)
\(\int e^x dx = e^x + C\)
\(\int a^x dx = \frac{a^x}{\ln a} + C\)
🟡 ỨNG DỤNG TÍCH PHÂN ▼
Diện tích hình phẳng
\(S = \int_a^b |f(x)-g(x)| dx\)
Thể tích tròn xoay (Ox)
\(V = \pi \int_a^b [f(x)]^2 dx\)
Thể tích vật thể (biết S(x))
\(V = \int_a^b S(x) dx\)
Thể tích tròn xoay (Oy)
\(V = \pi \int_{y_1}^{y_2} [x(y)]^2 dy\)
